Regulamin typowań
Opłata za udział
Za każdy mecz gracz wpłaca do puli 3 zł (dorosły) lub 1 zł (dziecko). Opłata naliczana jest automatycznie przy wysłaniu typu.
Deadline typowania
Typ należy przesłać najpóźniej 2 godziny przed meczem. Portal automatycznie blokuje możliwość typowania po tym terminie.
Co typujemy
Typujemy dokładny wynik meczu (np. 2:1). Tylko identyczne trafienie daje nagrodę — wynik 1:0 ≠ 2:0.
Podział puli
Jedno trafienie → cała pula dla zwycięzcy. Kilka trafień → pula dzielona do pełnych złotych między zwycięzców. Reszta groszy i ewentualna nadwyżka przechodzi do puli następnego meczu.
Pauza za spóźnienie
Brak typu lub typ po terminie = pauza. Pauzujący gracz nie płaci za ten mecz, ale nie może wrócić do typowania dopóki cała pula z meczów, które opuścił, nie zostanie rozdzielona między zwycięzców.
Automatyczne wyniki
Wyniki pobierane są automatycznie z API co 60 sekund.

Integracja football-data.org
Dane meczów MŚ 2026 pobierane są automatycznie co 60 sekund. Klucz API wpisz w pliku .env na serwerze. Darmowy plan (Free Tier) wystarczy na cały turniej.
GET /v4/competitions/WC/matches
GET /v4/competitions/WC/matches?status=LIVE
→ Rejestracja: football-data.org/client/register
Opisy AI (Anthropic Claude)
Po zakończeniu każdego meczu Claude automatycznie generuje podsumowanie w języku polskim: strzelcy, minuty goli, ciekawostki. Klucz API wpisz w .env jako ANTHROPIC_API_KEY.
